# 🕵️‍♂️ Secret Search Detective Agency Website

## 📌 Description

This is a modern, responsive website for a **Detective Agency** built using HTML, Tailwind CSS, and JavaScript.
The website provides information about services, features, contact details, and allows users to easily connect via call or WhatsApp.

---

## 🚀 Features

* 📱 Fully Responsive Design (Mobile + Desktop)
* 🧭 Navigation Bar with Mobile Menu Toggle
* 🎯 Hero Section with Call-to-Action
* 🛠️ Services Section (Surveillance, Cyber Crime, etc.)
* 📊 Features / Why Choose Us Section
* ❓ FAQ Section with Expand/Collapse functionality
* 📞 Contact Section with Call & WhatsApp Integration
* 📩 Contact Form UI
* 🧾 Footer with Quick Links & Social Icons

---

## 🛠️ Technologies Used

* HTML5
* Tailwind CSS
* JavaScript
* Font Awesome (Icons)
